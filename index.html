Security

Default: museum123. Unlock before changing anything.
Slot 0 is a fixed system beacon (not editable). “Save & Restart” writes flash and reboots — the device will disconnect (expected).
Supports http(s):// (www.) prefixes and .com suffix. Keep short (≤17 chars after encoding).
Broadcast as manufacturer-specific data.
“Connectable” off = this slot advertises non-connectable. The config channel stays connectable so you can always reach this console. Writes go to RAM — use General → “Save & Restart” to persist.
